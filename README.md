# 🧪 Experimental Lab

Welcome to my personal collection of experimental projects. Each project lives in its own folder with its own README, structure, and functionality. This repository is designed to showcase small-scale, local-first, and system-oriented projects. More projects will be added over time.

## 📂 Projects

### 🎵 LYRICSPlayer

A local music player with synchronized lyrics. Fully offline, plays your MP3 files with time-synced lyrics.

- 📁 **Folder:** [`lyrics_player/`](https://github.com/Wael-06/experimental-lab/tree/main/lyrics_player)
- ➡️ **Details:** [See `lyrics_player/README.md` for full details.](https://github.com/Wael-06/experimental-lab/blob/main/lyrics_player/README.md)

### 📄 Report PDF Generator

Deterministic, local-first PDF generation system for structured academic submissions. Turns screenshots and metadata into professional PDFs.

- 📁 **Folder:** [`report_pdf_generator/`](https://github.com/Wael-06/experimental-lab/tree/main/report_pdf_generator)
- ➡️ **Details:** [See `report_pdf_generator/README.md` for full details.](https://github.com/Wael-06/experimental-lab/blob/main/report_pdf_generator/README.md)
